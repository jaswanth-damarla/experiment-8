# 🚀 Experiment 8: Spring Boot REST API with MySQL

## 📌 Overview
This experiment demonstrates the development of a **RESTful API** using Spring Boot with MySQL database integration.  
The application performs basic CRUD operations for managing student data.

---

## 🛠️ Tech Stack
- Java 21
- Spring Boot
- Spring Web (REST API)
- Spring Data JPA
- MySQL
- Maven
- Postman (for API testing)

---

## 📂 Project Structure:
```
src/
└── main/
    ├── java/
    │   └── com.aml3a.rest.api/
    │       ├── controller/
    │       ├── service/
    │       ├── repository/
    │       └── model/
    └── resources/
        └── application.properties
```



---

## ⚙️ Features
- Create Student
- Retrieve Students
- Update Student
- Delete Student
- Store data in MySQL database
- RESTful endpoints tested using Postman

---

## 🗄️ Database Setup
- Database Name: `fs`
- Table: `student`
